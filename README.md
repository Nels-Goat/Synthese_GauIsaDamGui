# Synthese_GauIsaDamGui

Idées pour le projet : 

  

Roguelike  
Style Vampire Survivor 
Thème : Magie 

  

Armes : Arc, Bâton de mage, Corps à corps (3 sprites différents pour chaque armes (upgrades)) 
Habiletés globales : Dash, Soins, Buff de dégâts, Buff de vitesse, Buff max life  
Arc : Multishot,  
Baton : Buff de zone  
Corps à corps : Buff de range 

  

Habileté mystère :  

Positif : Wipe de map avec un dragon (Habileté) 
Négatif : Reset de tout (Habiletés, armes) en gardant les niveaux, les waves de mobs 

  

Mobs : Witches -> spawn des squelettes, Gobelins à lance, Dragons, Boss (à revoir)
